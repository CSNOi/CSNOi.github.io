# 点集拓扑

## 拓扑空间

拓扑空间是由集族X于在X上的拓扑共同生成的空间，记为（X，）

可数性公理
C1可数性公理：空间 $X$ 中的每一个点，都拥有一个可数的邻域基。
C2可数性公理：X空间中的拓扑T中的每一个元素都可以用同一组基表示。

分离公理
T0分离公理：当x不等于y，能找到只包含其中一个点的开集
T1分离公理：当x不等于y时，能找到只包含其中一个点的开集
T2分离公理：当x不等于y时，能找到不交的开集使得x，y分别属于其中
T3分离公理：满足T1空间的X，给定一个闭集 $A$ 和一个在它外面的点 $x$，有两个不交的开集可以分别包含它们。
T4分离公理：满足T1空间的X，给定两个不交的闭集 $A$ $B$，有两个不交的开集可以分别包含它们。
